callviz

Python call graph visualization for recursive functions. Provides a decorator to instrument the target function and pygraphviz to render the recursion visualization.

usage##

  1. Use the @viz decorator to instrument the recursive function.
  2. Render the recursion with > callgraph.render("outfile.png")

The file type is derived from the file name. Supported types include .dot (graphviz dot file), .png (png image), .svg (vector graphic)
